Vue 高阶组件三剑客

2019-10-05

让我们直接进入正题，这三剑客就是指 $props、$attrs、$listeners。

API 定义

先看官方文档定义然后再慢慢展开：

vm.$props: 当前组件接收到的 props 对象。
vm.$attrs: 包含了父作用域中不作为 prop 被识别 (且获取) 的特性绑定 (class 和 style 除外)。
vm.$listeners: 包含了父作用域中的 (不含 .native 修饰器的) v-on 事件监听器。
